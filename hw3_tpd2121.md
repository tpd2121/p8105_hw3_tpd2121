``` r
#joining the dataset
complete_acc = 
  full_join(demography, acc_data, by = "SEQN") %>% 
  filter(age >21) %>% 
  mutate(
    SEQN = as.numeric(SEQN),
    sex = as.factor(sex),
    age = as.numeric(age),
    BMI = as.double(BMI),
    education = as.factor(education)
    #mutate as.dbl for the min1:min94
  )
```

Creating table and visualization for gender, education, and age

